# Changelog

All notable changes to `DevPulse IDE` will be documented in this file.

## [0.1.0] - 2026-03-21

- Initial VS Code extension release.
- Added DevPulse sidebar view and full panel view.
- Added commands for sending editor context, analyzing selected code, copying context, refreshing, and opening web app.
- Added status bar launcher and editor selection shortcut.
- Added IDE-to-webapp bridge messaging support.
